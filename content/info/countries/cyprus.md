# Кипр

Чемпионаты Кипра проводятся с 2017 года. Первое время на острове проводились чемпионаты по спортивному «Что? Где? Когда?», но затем организаторы сменились и теперь чемпионаты проводятся по другой дисциплине — командной викторине с раундами по минуте.
